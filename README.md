解决的问题
----------
      为了更好地满足社区居家养老工作的需要，提高管理服务水平，本课题主要设计一个社区居家养老服务平台，实现居家养
      老信息发布；居家养老人员信息管理；居家养老服务预约；订餐、订菜；每天健康信息报告等服务的管理。
